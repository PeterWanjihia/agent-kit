agent-kit quickstart
$ go install github.com/ronexlemon/agent-kit/cmd/agent-kit@latest

$ agent-kit create \
  --name "Celo Assistant Agent" \
  --service MCP:https://agent.example/mcp \
  --service A2A:https://agent.example/a2a

$ agent-kit publish ./agent.json --provider pinata
✓ Metadata pinned to ipfs://bafy...

$ agent-kit register --network celo --metadata ipfs://bafy...
✓ Registered agent ID: 1

$ agent-kit resolve eip155:42220:0xRegistry:1
✓ services: MCP, A2A

Why it exists

Agents need more than wallets.

Useful on-chain agents need identity, metadata, service discovery, verifiable ownership, and reputation. agent-kit packages those building blocks into a Go SDK and CLI that other Celo builders can reuse.

SDKReusable Go packages for metadata, registry, resolver, network, and reputation workflows.
CLIA practical developer path for creating, publishing, registering, and resolving agents.
CeloDesigned around mainnet proof, agent references, explorer links, and public documentation.

Grant fit
